## Post

3 weeks ago I shipped an open source macOS app for developers. Here is what actually happened.

---

The problem started during my own Claude Code sessions. Scratch context accumulates fast: decisions made mid-session, files to revisit, warnings to not forget, temporary TODOs. None of it has a home. It lives in your head or gets lost in terminal scroll. By the time Claude loses context, so do you.

So I built Clause. A floating SwiftUI window that acts as an MCP server. Both you and Claude can read and write notes in real time during a session. When the session ends, the notes disappear. Ephemeral by design, minimal by default.
