**Tweet 2 (Problem)**

During a Claude Code session, scratch context piles up fast. Decisions made, warnings to track, TODOs, file bookmarks. None of it has a home. It lives in your head or gets buried in terminal scroll.

---

**Tweet 3 (Solution)**

Clause is a floating macOS window connected to Claude via MCP. Both you and Claude can read and write notes in real time. When the session ends, notes disappear. Ephemeral by design.

---

**Tweet 4 (Tech)**

Built with Swift 6 and SwiftUI. Two-process architecture: clause-mcp runs as a stdio MCP server, Clause.app is the floating window UI. They talk over a Unix domain socket. Open source, MIT licensed.
